# I-am-Super
Super Resolution Framework with Pytorch

## Environment
* Python 3.6 [+]
* Pytorch 1.0
* Ubuntu 16.04
* CUDA 8.0 [+]

## Dataset
* Vid4
* SPMCS
* Vimeo-90k
